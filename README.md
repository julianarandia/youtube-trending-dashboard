# YouTube Trending Dashboard

## Project Objective
The objective was to provide planning managers, Melanie and Ashok, with a tool to rapidly analyze YouTube video trends to determine which content deserves attention for their marketing strategy, effectively **transitioning content investment from guesswork to data-informed decisions**.

## About the Dataset
This project utilizes a raw dataset containing global daily trending YouTube video statistics. The data flow is designed to mimic an API connection, receiving daily updates every 24 hours (midnight UTC) from a core YouTube database.

## Conclusions
The project's central solution is an **interactive and automated dashboard** designed for planning managers, providing **daily consultation for strategic decision-making** regarding YouTube content investment. This tool successfully demonstrates the ability to connect directly to source-of-truth data for real-time reporting.
